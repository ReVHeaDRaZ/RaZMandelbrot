# RaZMandelbrot
A simple mandelbrot viewer with pan and zoom using SFML

# Controls

|  |  |
| ------------- | ------------- |
| <b>LMB-</b>           | Zoom In  |
| <b>RMB-</b>           | Zoom Out  |
| <b>UP and DOWN-</b>   | Pan UP and DOWN  |
| <b>LEFT and RIGHT-</b> | PAN LEFT and RIGHT |
| <b>PAGEUP and PAGEDOWN-</b> | Change amount of Iterations |
